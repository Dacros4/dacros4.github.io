[CN] Builder Server

欢迎你！你正在[CN] Builder Server上游玩！

这个服务器只允许游戏时间50小时以上的玩家进入，如果你还没满，请先去其他服务器！

 一、管理员

  基本每时每刻都有的，有啥事就发@+举报消息

  没有的话就自己克制一下，不要炸服，后台可以看见的，抄下steamid就是ban

  服主是LittleSDY，名字是纯白色的

 二、规则

  服务器里有时候会有玩家组织pvp，如果你不参与，就到别处去，请勿打扰，不然就kick了

  pvp禁止god伤人（弟弟行为），也不要开透视，飞行啥的，公平一点~

  不要在人家造车或者其他东西的时候用枪打，真的很烦，如果对方挂机了然后你又有事情可以开一两枪(

  不要用刷东西还不冻结，不用的东西就按Z清清掉，每天炸服也吃不消

  请勿使用大规模的爆炸e2/副本，或者是能够影响到他人创作的，发现就直接kick/ban

  (口嗨也要有个度，禁止刷屏)

 三、注意事项

  服务器有比较多的插件，如果出现error或者自黑格，请挂载CS:S并订阅所有插件
